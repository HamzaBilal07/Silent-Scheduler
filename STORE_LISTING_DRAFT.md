# Store Listing Draft

## App Name

Silent Scheduler

## Short Description

Automatically silence your phone for prayers, lectures, meetings, and events.

## Full Description

Silent Scheduler helps you keep your phone silent during important moments and automatically restores sound when the event ends.

Create schedules for prayers, lectures, meetings, study sessions, work hours, Jummah, or any custom event. Each event has a start time, end time, enable switch, and repeat options. At the start time, Silent Scheduler enables Do Not Disturb. At the end time, it restores the previous sound mode.

The app also includes prayer timing support. With location permission, Silent Scheduler can load daily prayer timings and sunrise from the internet according to your location. Hanafi and Jafria fiqah options are available in Settings.

## Key Features

- Simple alarm-style event list
- Unlimited custom events
- Custom repeat days, daily repeat, Friday-only, and one-time schedules
- Quick DND timer for 15, 30, or 60 minutes
- Automatic Do Not Disturb scheduling
- Previous sound mode restore
- Automatic prayer timings using location and internet
- Hanafi and Jafria fiqah options
- Sunrise time on the Events page
- Daily Quran ayah with Urdu translation
- Offline ayah library for no-internet use
- Dedicated Prayer Timings page
- Swipe down to refresh prayer timings
- Reboot-safe scheduling
- Local schedule and settings storage
- Simple and familiar Android interface

## Permissions

Silent Scheduler requests only the access required for its main features:

- Do Not Disturb access: used to silence the phone during scheduled events.
- Exact alarm access: used so event start and end actions happen on time.
- Notification permission: used for reminders and important status messages.
- Location permission: used only to calculate daily prayer timings and sunrise.
- Internet access: used to load prayer timing data.

The app does not use ads and does not sell personal data.
