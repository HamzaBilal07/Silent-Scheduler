# Store Listing Draft

## App Name

Silent Scheduler

## Short Description

Silence your phone for prayers, lectures, and events automatically.

## Full Description

Silent Scheduler helps you keep your phone silent during prayers, lectures, meetings, and other events without forgetting to turn sound back on.

Create event schedules manually or let the app update daily prayer times using your phone location. Each event has a start time, end time, repeat option, and enable switch. At the start time, the app turns on Do Not Disturb. At the end time, it restores the previous sound mode.

Features:

- Simple alarm-style event list
- Empty first-run list so users add only their own event times
- Quick DND timer for instant 15, 30, or 60 minute silence
- Manual schedules for lectures, meetings, prayers, and more
- Automatic daily prayer timings using location and internet
- Hanafi and Jafria fiqah options
- Sunrise time on the Events page
- Daily Quran ayah with Urdu translation
- Dedicated Prayer timings page
- Swipe down to refresh prayer timings
- Daily and Friday-only repeat options
- Custom weekday selection, including Monday-Friday schedules
- Do Not Disturb automation
- Previous sound mode restore
- Reboot rescheduling
- Local schedule and settings storage

Required access:

- Do Not Disturb access is used only to silence the phone during scheduled events.
- Exact alarm access is used so event start and end actions run on time.
- Location access is used only to calculate daily prayer timings and sunrise.
- Notification access is used for warnings and optional reminders.
